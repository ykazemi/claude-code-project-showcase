# Submit Your Claude Code Workshop Project

Each trainee or team should submit one project post.

## What to include

Your post should include:

1. Project title
2. Team members
3. Short summary
4. Problem you are solving
5. What you built
6. Tools used
7. How you used Claude Code
8. Screenshots or demo GIF
9. GitHub repository link
10. What you learned
11. Future improvements

## How to submit

1. Fork this repository.
2. Go to the `_posts` folder.
3. Create a new Markdown file.
4. Name your file using this format:

```text
YYYY-MM-DD-project-title.md
```

For example:

```text
2026-05-25-ai-study-buddy.md
```

5. Copy the project template below.
6. Fill it in.
7. Add your screenshots to:

```text
assets/images/YOUR-PROJECT-NAME/
```

8. Open a pull request.
