## Project Overview

AI Study Buddy is a simple learning assistant that helps students turn their notes into practice questions.

## Problem

Students often have notes but do not know how to turn them into active recall practice.

## What We Built

We built a Streamlit app where users can paste their notes and generate quiz questions.

## How We Used Claude Code

We used Claude Code to help us:

- Set up the project structure
- Debug Python errors
- Improve the user interface
- Refactor repeated code
- Write a README
