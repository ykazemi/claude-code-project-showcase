## Project Overview

How Fast is a customized Python/Pygame RPG built from an open-source Zelda-style game environment. We used the original project mainly as a starting environment, then redesigned the game identity, created a new user story, added custom character ideas, expanded the weapon concept, introduced quest tracking, and added more gameplay flavor.

The project focuses on turning a familiar RPG base into a more original, fast-paced survival quest.

## Problem

The original game environment was functional, but was majorly a environment. We wanted to make it feel more like our own game by giving it a clearer story, stronger player motivation, new character direction, and more interactive gameplay.

## What We Built

We redesigned the game into **How Fast**, a speed-run survival quest set in a cursed forest called Clockwood.

The main additions include:

- A new game title and theme
- A pregame user story
- Character selection planning with three runners
- New character personalities and stat concepts
- Added weapon and combat flavor
- A quest tracker for defeating enemies
- A planned Enter-to-shoot projectile mechanic
- A cleaner plan for extending the game safely

## How We Used Claude Code

We used Claude Code to help us:

- Understand the existing Python/Pygame project structure
- Identify which parts of the repo controlled the environment, player, enemies, UI, and settings
- Plan safe changes without rewriting the whole game
- Design the new game flow: start → story → character selection → game → death
- Draft the user story and gameplay concept
- Plan character selection, quest tracking, weapons, and shooting mechanics
- Debug and organize the project changes step by step
