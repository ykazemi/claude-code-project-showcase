## Project Overview

How Fast is a customized Python/Pygame RPG built from an open-source Zelda-style game project. We reworked the game identity, added a pregame story, introduced character selection, created a quest tracker, and planned a new shooting mechanic so the player can attack enemies using the Enter key.

The project focuses on making a small but playable game feel more original, fast-paced, and demo-ready.

## Problem

The original game was functional, but it still felt like a template project. We wanted to turn it into something with a clearer identity, a stronger opening experience, and more interactive gameplay features that could be extended further.

## What We Built

We redesigned the game into **How Fast**, a speed-run survival quest set in a cursed forest called Clockwood.

The main additions include:

- A new game title and theme
- A pregame story screen
- A character selection menu with three runners
- A quest tracker for defeating enemies
- A planned Enter-to-shoot projectile mechanic
- A cleaner implementation plan for extending the game safely

## How We Used Claude Code

We used Claude Code to help us:

- Understand the existing Python/Pygame project structure
- Plan the safest files to modify
- Design the new game flow: start → story → character selection → game → death
- Create a feature spec for renaming, story, character selection, quests, and shooting
- Generate implementation prompts for each phase
- Think through testing steps and risks before editing the code
